제 7회 (도전 난이도) — 정답 및 풀이
채점 팁 ① 서술형은 공식 인용 + 중간 식 + 결론 3요소가 있어야 만점 ② 인수분해 경우 열거 문항은 빠짐 1건당 −2점 ③ 무리수 판별 문항은 근거가 빠지면 값이 맞아도 절반 감점
1 무리수를 수직선 위에 나타내기 25점
(1) [10점]
정사각형 ABCD에서 한 변의 길이가 1이므로 △ABC는 직각삼각형이고 두 변이 각각 1이다. 피타고라스 정리에 의해 AC2 = AB2 + BC2 = 12 + 12 = 2 따라서 AC = √2 (마찬가지로 BD = √2).
답AC = √2
(2) [15점]
점 A는 √3, 점 B는 √3 + 1에 놓여 있다. AC = BD = √2이므로 원과 수직선의 교점은 다음과 같다. P = √3 − √2 (A 중심, 작은 값) Q = √3 + √2 (A 중심, 큰 값) R = (√3 + 1) − √2 (B 중심, 작은 값) S = (√3 + 1) + √2 (B 중심, 큰 값) 따라서 a = √3 − √2, b = √3 + √2, c = √3 + 1 − √2, d = √3 + 1 + √2.
a·b = (√3 − √2)(√3 + √2) = (√3)2 − (√2)2 = 3 − 2 = 1
c·d = (√3 + 1 − √2)(√3 + 1 + √2) = (√3 + 1)2 − (√2)2 = (3 + 2√3 + 1) − 2 = 2 + 2√3 ∴ a·b + c·d = 1 + 2 + 2√3 = 3 + 2√3
답a·b + c·d = 3 + 2√3
포인트: (A중심·반지름 √2), (B중심·반지름 √2) 두 쌍을 (a+b)(a−b) = a²−b² 꼴로 묶으면 계산량이 급감. 곱셈공식 단원의 선행 활용을 유도하는 문항.
2 제곱근의 계산 25점
(1) [10점]
세로 3√2가 가로 5√2보다 짧으므로 첫 정사각형 A의 변은 세로에 맞춰 3√2 cm. 남은 공간은 (5√2 − 3√2) × 3√2 = 2√2 × 3√2. 이제 가로 2√2가 세로 3√2보다 짧으므로 두 번째 정사각형 B의 변은 2√2 cm, 남은 공간은 2√2 × (3√2 − 2√2) = 2√2 × √2. 마지막으로 세 번째 정사각형 C의 변은 √2 cm.
세 변의 곱 = 3√2 · 2√2 · √2 = 6 · (√2)3 = 6 · 2√2 = 12√2.
답A = 3√2, B = 2√2, C = √2, 곱 = 12√2
(2) [15점]
① 사분원 넓이 = π r² / 4 S_A = π(3√2)²/4 = 18π/4 = 9π/2 S_B = π(2√2)²/4 = 8π/4 = 2π S_C = π(√2)²/4 = 2π/4 = π/2 ② 직사각형 넓이 = 5√2 · 3√2 = 15 · 2 = 30. 사분원 합 = 9π/2 + 2π + π/2 = (9π + 4π + π)/2 = 14π/2 = 7π. 나머지 부분 = 30 − 7π (cm²) ③ 30 − 7π 를 a + bπ 꼴로 보면 a = 30, b = −7. π는 무리수이므로 7π도 무리수이고, (유리수 30) − (무리수 7π)는 무리수이다. 따라서 나머지 부분의 넓이는 무리수.
답① 9π/2, 2π, π/2 ② 30 − 7π ③ a = 30, b = −7, 무리수
3 대수막대를 이용한 인수분해 25점
(1) [10점] — 1 막대 16개
x2 + kx + 16 이 (x + m)(x + n) 꼴로 인수분해되려면 mn = 16, m + n = k. m, n은 양의 정수.
(m, n)x막대 개수 k인수분해
(1, 16)17(x + 1)(x + 16)
(2, 8)10(x + 2)(x + 8)
(4, 4)8(x + 4)² [완전제곱]
→ 총 3가지.
(2) [15점]
① 1 막대 25개: mn = 25
(m, n)k인수분해
(1, 25)26(x + 1)(x + 25)
(5, 5)10(x + 5)² [완전제곱]
→ 2가지.

② 1 막대 36개: mn = 36
(m, n)k인수분해
(1, 36)37(x + 1)(x + 36)
(2, 18)20(x + 2)(x + 18)
(3, 12)15(x + 3)(x + 12)
(4, 9)13(x + 4)(x + 9)
(6, 6)12(x + 6)² [완전제곱]
→ 5가지.
답(1) 3가지 · (2)① 2가지 · (2)② 5가지 — 총 10가지
흔한 실수: "1+n" 형태(예: (1,16)/(1,25)/(1,36))를 "너무 뻔하다"고 누락, 또는 완전제곱 (4,4)/(5,5)/(6,6)을 "같은 수라 안 된다"고 제외. 둘 다 인정되는 경우이므로 반드시 포함해야 한다.
4 곱셈공식을 이용한 식의 계산 25점
(1) [10점]
두 반지름 R = √7 + √3, r = √7 − √3. 곱셈공식 (a+b)2 = a2 + 2ab + b2, (a−b)2 = a2 − 2ab + b2를 적용하면 R2 = (√7 + √3)2 = 7 + 2√21 + 3 = 10 + 2√21 r2 = (√7 − √3)2 = 7 − 2√21 + 3 = 10 − 2√21 ∴ R2 + r2 = (10 + 2√21) + (10 − 2√21) = 20 중간항 ±2√21 이 서로 소거되어 유리수 20으로 깔끔하게 정리된다.
답R2 + r2 = 20
채점 포인트: 직접 전개해서 20을 얻어도 좋지만, 문항이 명시적으로 곱셈공식을 요구했으므로 완전제곱 공식 2개를 풀이 중에 반드시 인용해야 만점. 학생이 "어느 공식을 쓸지"를 직접 판단하는 것까지 사고의 범위임.
(2) [15점]
① 색칠 부분의 넓이 = (큰 원 넓이) − (작은 원 넓이) = πR2 − πr2. 먼저 R + r = (√7 + √3) + (√7 − √3) = 2√7, R − r = (√7 + √3) − (√7 − √3) = 2√3. 곱셈공식 (a+b)(a−b) = a2 − b2을 적용하면 πR2 − πr2 = π(R2 − r2) = π(R + r)(R − r) = π · 2√7 · 2√3 = 4√21 π (cm2) ② k = 4√21
③ 21 = 3 · 7 로 완전제곱수가 아니므로 √21은 무리수. 유리수 4에 무리수 √21을 곱한 4√21 역시 무리수이다.
답① 4√21 π cm² ② k = 4√21 ③ 무리수
(1)에서는 완전제곱 공식 (a±b)² = a² ± 2ab + b², (2)에서는 차의 곱 공식 (a+b)(a−b) = a² − b² — 서로 다른 곱셈공식 두 개를 교차 사용해야 만점. 문항에서는 "곱셈공식을 이용하여"라고만 했으므로 어떤 공식을 쓸지 판단하는 것 자체가 평가 대상이다.
