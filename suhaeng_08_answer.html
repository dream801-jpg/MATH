제 8회 (도전 난이도) — 정답 및 풀이
학습 가이드 ① 서술형은 공식 인용 → 식 전개 → 결론 세 단계로 쓰자. ② 경우를 열거할 때는 작은 수부터 체계적으로 확인하면 빠뜨리지 않는다. ③ 무리수·유리수 판별은 값뿐 아니라 근거까지 함께 서술하자.
1 무리수를 수직선 위에 나타내기 25점
(1) [10점]
정사각형 ABCD의 한 변이 1이므로 △ABC는 직각삼각형이고 두 변이 각각 1이다. 피타고라스 정리에 의해 AC2 = AB2 + BC2 = 12 + 12 = 2 따라서 AC = √2 (마찬가지로 BD = √2).
답AC = √2
(2) [15점]
점 A는 √5, 점 B는 √5 + 1 에 놓여 있다. AC = BD = √2이므로 원과 수직선의 교점은 다음과 같다. P = √5 − √2 (A 중심, 작은 값) Q = √5 + √2 (A 중심, 큰 값) R = (√5 + 1) − √2 (B 중심, 작은 값) S = (√5 + 1) + √2 (B 중심, 큰 값) 따라서 a = √5 − √2, b = √5 + √2, c = √5 + 1 − √2, d = √5 + 1 + √2.
a·b = (√5 − √2)(√5 + √2) = (√5)2 − (√2)2 = 5 − 2 = 3
c·d = (√5 + 1 − √2)(√5 + 1 + √2) = (√5 + 1)2 − (√2)2 = (5 + 2√5 + 1) − 2 = 4 + 2√5 ∴ a·b + c·d = 3 + 4 + 2√5 = 7 + 2√5
답a·b + c·d = 7 + 2√5
💡 학습 포인트 — a·b와 c·d 모두 "두 수의 합과 차"를 곱하는 꼴이다. 곱셈공식 (a+b)(a−b) = a²−b² 를 알아보고 바로 적용하면 일일이 전개할 필요가 없다. 기준점이 √3이든 √5든 √6이든, 구조가 같으면 같은 방식으로 풀린다.
2 제곱근의 계산 25점
(1) [10점]
세로 5√2가 가로 9√2보다 짧으므로 첫 정사각형 A의 변은 5√2 cm. 남은 공간은 (9√2 − 5√2) × 5√2 = 4√2 × 5√2. 이 중 짧은 변은 4√2이므로 두 번째 정사각형 B의 변은 4√2 cm, 남은 공간은 4√2 × (5√2 − 4√2) = 4√2 × √2. 이 중 짧은 변이 √2이므로 세 번째 정사각형 C의 변은 √2 cm.
세 변의 곱 = 5√2 · 4√2 · √2 = 20 · (√2)3 = 20 · 2√2 = 40√2.
답A = 5√2, B = 4√2, C = √2, 곱 = 40√2
(2) [15점]
① 사분원 넓이 = π r² / 4 S_A = π(5√2)²/4 = 50π/4 = 25π/2 S_B = π(4√2)²/4 = 32π/4 = 8π S_C = π(√2)²/4 = 2π/4 = π/2 ② 직사각형 넓이 = 9√2 · 5√2 = 45 · 2 = 90. 사분원 합 = 25π/2 + 8π + π/2 = (25π + 16π + π)/2 = 42π/2 = 21π. 나머지 부분 = 90 − 21π (cm²) ③ 90 − 21π 를 a + bπ 꼴로 보면 a = 90, b = −21. π는 무리수이므로 21π도 무리수이고, (유리수 90) − (무리수 21π) 는 무리수이다.
답① 25π/2, 8π, π/2 ② 90 − 21π ③ a = 90, b = −21, 무리수
3 대수막대를 이용한 인수분해 25점
(1) [10점] — 1 막대 20개
x2 + kx + 20 이 (x + m)(x + n) 꼴로 인수분해되려면 mn = 20, m + n = k. m, n은 양의 정수.
(m, n)k인수분해
(1, 20)21(x + 1)(x + 20)
(2, 10)12(x + 2)(x + 10)
(4, 5)9(x + 4)(x + 5)
→ 총 3가지.
(2) [15점]
① 1 막대 49개: mn = 49
(m, n)k인수분해
(1, 49)50(x + 1)(x + 49)
(7, 7)14(x + 7)² [완전제곱]
→ 2가지.

② 1 막대 100개: mn = 100
(m, n)k인수분해
(1, 100)101(x + 1)(x + 100)
(2, 50)52(x + 2)(x + 50)
(4, 25)29(x + 4)(x + 25)
(5, 20)25(x + 5)(x + 20)
(10, 10)20(x + 10)² [완전제곱]
→ 5가지.
답(1) 3가지 · (2)① 2가지 · (2)② 5가지 — 총 10가지
💡 학습 포인트 — 100의 경우 (5, 20)이나 (4, 25)처럼 "딱 보기엔 안 나올 것 같은 쌍"도 있다. 약수 짝을 빠뜨리지 않으려면 1부터 √N 까지 차례로 나눠보며 약수 쌍을 완성하는 것이 안전한 방법. N이 완전제곱수(49, 100)라면 √N 이 자기 자신과 짝을 이루는 완전제곱식 (x+√N)² 도 반드시 나온다.
4 곱셈공식을 이용한 식의 계산 25점
(1) [10점]
두 반지름 R = √11 + √5, r = √11 − √5. 곱셈공식 (a+b)2 = a2 + 2ab + b2, (a−b)2 = a2 − 2ab + b2 를 적용하면 R2 = (√11 + √5)2 = 11 + 2√55 + 5 = 16 + 2√55 r2 = (√11 − √5)2 = 11 − 2√55 + 5 = 16 − 2√55 ∴ R2 + r2 = (16 + 2√55) + (16 − 2√55) = 32 중간항 ±2√55 가 서로 소거되어 유리수 32로 깔끔하게 정리된다.
답R2 + r2 = 32
💡 학습 포인트 — R², r² 를 각각 전개한 뒤 더하면 중간항이 서로 반대부호라서 소거된다. 이 현상은 제곱근 계산에서 자주 나오는 중요한 감각이다. (1)에서 사용한 공식은 (2)에서 쓰는 공식과 다르다는 점도 기억해 두자.
(2) [15점]
① 색칠 부분의 넓이 = (큰 원 넓이) − (작은 원 넓이) = πR2 − πr2. 먼저 R + r = (√11 + √5) + (√11 − √5) = 2√11, R − r = (√11 + √5) − (√11 − √5) = 2√5. 곱셈공식 (a+b)(a−b) = a2 − b2 을 적용하면 πR2 − πr2 = π(R + r)(R − r) = π · 2√11 · 2√5 = 4√55 π (cm2) ② k = 4√55
③ 55 = 5 · 11 로 완전제곱수가 아니므로 √55는 무리수. 유리수 4에 무리수 √55를 곱한 4√55 역시 무리수이다.
답① 4√55 π cm² ② k = 4√55 ③ 무리수
💡 학습 포인트 — 같은 동심원 상황에서 (1)은 완전제곱 공식, (2)는 차의 곱 공식을 썼다. 문제는 "곱셈공식을 이용하여"라고만 했으므로 어떤 공식을 쓸지 스스로 판단하는 것도 평가의 한 축. 목표(R²+r² 구하기 vs R²−r² 구하기)에 맞춰 공식을 선택하는 연습이다.
