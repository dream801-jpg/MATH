제 9회 (도전 난이도) — 정답 및 풀이
학습 가이드 ① 서술형은 공식 인용 → 식 전개 → 결론 세 단계로 쓰자. ② 경우를 열거할 때는 작은 수부터 체계적으로 확인하면 빠뜨리지 않는다. ③ 무리수·유리수 판별은 값뿐 아니라 근거까지 함께 서술하자.
1 무리수를 수직선 위에 나타내기 25점
(1) [10점]
정사각형 ABCD의 한 변이 1이므로 △ABC는 직각삼각형이고 두 변이 각각 1이다. 피타고라스 정리에 의해 AC2 = AB2 + BC2 = 12 + 12 = 2 따라서 AC = √2 (마찬가지로 BD = √2).
답AC = √2
(2) [15점]
점 A는 √6, 점 B는 √6 + 1에 놓여 있다. AC = BD = √2이므로 P = √6 − √2 (A 중심, 작은 값) Q = √6 + √2 (A 중심, 큰 값) R = (√6 + 1) − √2 (B 중심, 작은 값) S = (√6 + 1) + √2 (B 중심, 큰 값) 따라서 a = √6 − √2, b = √6 + √2, c = √6 + 1 − √2, d = √6 + 1 + √2.
a·b = (√6 − √2)(√6 + √2) = (√6)2 − (√2)2 = 6 − 2 = 4
c·d = (√6 + 1 − √2)(√6 + 1 + √2) = (√6 + 1)2 − (√2)2 = (6 + 2√6 + 1) − 2 = 5 + 2√6 ∴ a·b + c·d = 4 + 5 + 2√6 = 9 + 2√6
답a·b + c·d = 9 + 2√6
💡 학습 포인트 — 7회, 8회, 9회의 1번은 기준점만 √3, √5, √6으로 바뀌지 구조는 같다. 한 번 이 구조(합·차의 곱 → 유리수 결론)를 익히면 기준점이 뭐든 기계적으로 풀린다. "구조를 파악하는 습관"이 수학 성적을 결정한다.
2 제곱근의 계산 25점
(1) [10점]
세로 4√2가 가로 7√2보다 짧으므로 첫 정사각형 A의 변은 4√2 cm. 남은 공간은 (7√2 − 4√2) × 4√2 = 3√2 × 4√2. 짧은 변이 3√2이므로 두 번째 정사각형 B의 변은 3√2 cm, 남은 공간은 3√2 × (4√2 − 3√2) = 3√2 × √2. 짧은 변이 √2이므로 세 번째 정사각형 C의 변은 √2 cm.
세 변의 곱 = 4√2 · 3√2 · √2 = 12 · (√2)3 = 12 · 2√2 = 24√2.
답A = 4√2, B = 3√2, C = √2, 곱 = 24√2
(2) [15점]
① 사분원 넓이 = π r² / 4 S_A = π(4√2)²/4 = 32π/4 = 8π S_B = π(3√2)²/4 = 18π/4 = 9π/2 S_C = π(√2)²/4 = 2π/4 = π/2 ② 직사각형 넓이 = 7√2 · 4√2 = 28 · 2 = 56. 사분원 합 = 8π + 9π/2 + π/2 = (16π + 9π + π)/2 = 26π/2 = 13π. 나머지 부분 = 56 − 13π (cm²) ③ 56 − 13π 를 a + bπ 꼴로 보면 a = 56, b = −13. π는 무리수이므로 13π도 무리수이고, (유리수 56) − (무리수 13π) 는 무리수이다.
답① 8π, 9π/2, π/2 ② 56 − 13π ③ a = 56, b = −13, 무리수
3 대수막대를 이용한 인수분해 25점
(1) [10점] — 1 막대 32개
x2 + kx + 32 이 (x + m)(x + n) 꼴로 인수분해되려면 mn = 32, m + n = k. m, n은 양의 정수.
(m, n)k인수분해
(1, 32)33(x + 1)(x + 32)
(2, 16)18(x + 2)(x + 16)
(4, 8)12(x + 4)(x + 8)
→ 총 3가지. (32는 완전제곱수가 아니므로 완전제곱식 형태는 나오지 않는다.)
(2) [15점]
① 1 막대 64개: mn = 64
(m, n)k인수분해
(1, 64)65(x + 1)(x + 64)
(2, 32)34(x + 2)(x + 32)
(4, 16)20(x + 4)(x + 16)
(8, 8)16(x + 8)² [완전제곱]
→ 4가지.

② 1 막대 81개: mn = 81
(m, n)k인수분해
(1, 81)82(x + 1)(x + 81)
(3, 27)30(x + 3)(x + 27)
(9, 9)18(x + 9)² [완전제곱]
→ 3가지.
답(1) 3가지 · (2)① 4가지 · (2)② 3가지 — 총 10가지
💡 학습 포인트 — 81은 3과 9 만 약수 쌍을 만든다(3×27, 9×9). "약수의 개수가 적으면 경우의 수도 적다"는 감각을 키우자. 또 완전제곱수 64, 81은 반드시 완전제곱식 (x+8)², (x+9)²를 포함한다는 점이 핵심.
4 곱셈공식을 이용한 식의 계산 25점
(1) [10점]
두 반지름 R = 2√3 + √2, r = 2√3 − √2. 곱셈공식 (a+b)2 = a2 + 2ab + b2, (a−b)2 = a2 − 2ab + b2 를 적용하면 R2 = (2√3 + √2)2 = (2√3)2 + 2·2√3·√2 + (√2)2 = 12 + 4√6 + 2 = 14 + 4√6 r2 = (2√3 − √2)2 = (2√3)2 − 2·2√3·√2 + (√2)2 = 12 − 4√6 + 2 = 14 − 4√6 ∴ R2 + r2 = (14 + 4√6) + (14 − 4√6) = 28 중간항 ±4√6 이 서로 소거되어 유리수 28로 깔끔하게 정리된다.
답R2 + r2 = 28
💡 학습 포인트 — 이번 문항은 R = 2√3 + √2 처럼 계수가 있는 무리수를 다룬다. (2√3)² = 4·3 = 12 임을 정확히 계산하는 것이 첫 번째 관문. 2·2√3·√2 = 4√6 도 중요한 계산 연습이다.
(2) [15점]
① 색칠 부분의 넓이 = (큰 원 넓이) − (작은 원 넓이) = πR2 − πr2. 먼저 R + r = (2√3 + √2) + (2√3 − √2) = 4√3, R − r = (2√3 + √2) − (2√3 − √2) = 2√2. 곱셈공식 (a+b)(a−b) = a2 − b2 을 적용하면 πR2 − πr2 = π(R + r)(R − r) = π · 4√3 · 2√2 = 8√6 π (cm2) ② k = 8√6
③ 6 = 2 · 3 은 완전제곱수가 아니므로 √6은 무리수. 유리수 8에 무리수 √6을 곱한 8√6 역시 무리수이다.
답① 8√6 π cm² ② k = 8√6 ③ 무리수
💡 학습 포인트 — 7·8·9회 4번 모두 구조는 동일하다. 다만 9회는 R의 계수가 2로 달라서 √3·√2 = √6 이라는 제곱근의 곱까지 활용해야 한다. 4√3 · 2√2 = 8√6 로 변환하는 연습이 이 문항의 핵심이다.
